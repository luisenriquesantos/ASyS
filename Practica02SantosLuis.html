Operaciones de Señales:
Las operaciones de señales para sesplazamiento y escala empleadas en señales continuas, tambien puden aplicarse a señales de tiempo discreto con algunas modificaciones.
Considere una señal

xlabel ('n'); ylabel('x[n]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';

Problema lathi 3.2.3

Para la señal mostrada en la figura. P3.1-1b, grafique las siguientes señales.
a) x [ -n]
b) x [n+6 ]
c) x [ n-6]
d) x[3n]
e) x[n/3]
f) x[3-n]
f=@(n) n.*(0<n).*(n<4)+(-n+6).*(n>3).*(n<7);
figure
n=-3:8;
stem(n,f(n))
xlabel ('n'); ylabel('x[n]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';
title('Figura P3.1-1b')
axis ([-4 8,-1 4]);
Solucion:
a)
Para graficar la función x [ -n], se refleja la función original respecto al eje vertical, es decir, al valor n=1 se coloca en n=-1 conservando su altura, el valor n=2 se coloca en n=-2 conservando su altura, en consecuencia la grafica se ve como en la figura:
figure
n=-8:3;
stem(n,f(-n))
xlabel ('n'); ylabel('x[n]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';
title('Grafica x[-n]')
axis ([-8 4,-1 4]);
b)
Para la grafica de x [n+6 ], se toma la función x [n] y cada punto se desplaza seis unidades a la izquierda, conservando sus alturas.
figure
stem(n,f(n+6))
xlabel ('n'); ylabel('x[n+6]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';
title('Grafica x[n+6]')
axis ([-8 4,-1 4]);
c)
Para la grafica de x [n-6 ], se toma la función x [n] y cada punto se desplaza seis unidades a la derecha, conservando sus alturas.
figure
n=-1:16;
stem(n,f(n-6))
xlabel ('n'); ylabel('x[n-6]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';
title('Grafica x[n-6]')
axis ([-1 15,-1 4]);
d)
Para la grafica de x[3n], la funciòn se comprime horizontalmente en un factor de tres, debido a esta contracción, se pierde información de la señal.
figure
n=-1:16;
stem(n,f(3.*n))
xlabel ('n'); ylabel('x[3n]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';
title('Grafica x[3n]')
axis ([-1 3,-1 4]);
e)
Para la gráfica de x[n/3], la señal se expande horizontalmente en un factor de 3.
figure
n=-1:16;
stem(3*n,f(n))
xlabel ('n'); ylabel('x[n/3]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';
title('Grafica x[n/3]')
axis ([-1 17,-1 4]);
f)
Para graficar x[3-n], nos apoyamos de la grafica de x[-n] y la desplazamos tres unidades a la derecha.
figure
n=-4:6;
stem(n,f(-n+3))
xlabel ('n'); ylabel('x[3-n]'); grid;
ax=gca;
ax.XAxisLocation = 'origin';
ax.YAxisLocation = 'origin';%ax.Box = 'off';
title('Grafica x[-n+3]')
axis ([-4 6,-1 4]);
